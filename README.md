##  Overview

This project is an interactive **Power BI dashboard** designed to analyze student performance across multiple dimensions such as scores, attendance, behavior, subjects, and class distribution.
It provides actionable insights to help educators and stakeholders improve academic outcomes.


##  Key Features

*  **Performance Tracking** – Analyze average scores across subjects and classes
*  **Student-Level Insights** – View individual student scores and rankings
*  **Class & Section Analysis** – Compare performance across different classes
*  **Term-wise Trends** – Track score changes across multiple terms
*  **Attendance Monitoring** – Evaluate impact of attendance on performance
*  **Behavior Insights** – Understand behavioral patterns and their influence

---

##  Dataset Information

The dashboard uses structured datasets including:

* Student details (ID, Class, Gender)
* Scores by subject and term
* Attendance percentage
* Behavior categories
* Exam types and max scores



##  Dashboard Components

* **KPI Cards**

  * Total Students
  * Average Score
  * Attendance %

* **Visualizations**

  * Stacked bar chart (Scores by Class & Subject)
  * Line chart (Score trend by term)
  * Donut chart (Behavior distribution)
  * Table (Student-wise performance)

* **Filters / Slicers**

  * Class
  * Section
  * Subject
  * Term



##  Tools & Technologies

*  Power BI Desktop
*  CSV Data Sources
*  DAX (Data Analysis Expressions)


##  Insights Generated

* Students show **consistent but average performance (~50%)**
* High attendance (~90%) does not fully translate into high scores
* Minimal variation across subjects indicates uniform difficulty level
* Slight performance drop observed in later terms
* Behavior patterns may influence academic outcomes

##  Use Cases

* Academic performance monitoring
* School/college reporting dashboards
* Data visualization learning project
